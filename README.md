🌟 Landing Page Instasany

Projeto de **landing page animada** aplicada com **Next.js**, utilizando **GSAP** para animações e `use client` para renderização no lado do cliente.

<hr>

🚀 Tecnologias
- Next.js
- React
- GSAP
- TailwindCSS

<hr>

 🎯 Objetivo
Praticar criação de interfaces modernas, animações e organização de layout em aplicações React.

<hr>

🙌 Créditos

Layout e inspiração do projeto: CodeBoost
